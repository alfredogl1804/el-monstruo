# Brand DNA: Naranja Forja + Graphite + Acero

## Decisión

Se establece como restricción dura la paleta de colores canónica para El Monstruo: Naranja Forja (#F97316) como color primario, Graphite oscuro (#1C1917) y Acero medio (#A8A29E). El estilo visual debe seguir un enfoque de brutalismo industrial refinado, alineado con el arquetipo de marca Creador + Mago.

## Por qué

Esta paleta y estilo visual comunican la naturaleza técnica, poderosa y transformadora del orquestador IA. El brutalismo industrial refleja eficiencia y estructura, mientras que el arquetipo Creador + Mago subraya la capacidad de construir soluciones innovadoras y ejecutar magia tecnológica.

## Implicaciones

Cualquier interfaz, documento o material visual generado para El Monstruo debe adherirse estrictamente a estos códigos hexadecimales y estilo. No se permiten variaciones de color fuera de esta paleta canónica.
