# 2. PROBLEMAS DE COMPATIBILIDAD DETECTADOS

## a) PSU de 1000W para RTX 5090

Estado: SUFICIENTE PERO AJUSTADO

NVIDIA recomienda mínimo 1000W para RTX 5090 (TGP 575W)

Problema: Picos transitorios pueden alcanzar hasta 659W en juegos como Cyberpunk 2077

Recomendación: Actualizar a PSU de 1200W-1500W para headroom adecuado

## b) Motherboards compatibles con ATX 3.1

Requisito crítico: El motherboard debe soportar PCIe 5.0

Compatibilidad: Cualquier motherboard con slot PCIe x16 funciona, pero para máximo rendimiento se requiere PCIe 5.0

El estándar ATX 3.1 se refiere al PSU, no al motherboard

## c) 64GB DDR5 para D5 Render

Estado: SUFICIENTE

64GB es adecuado para la mayoría de escenas en D5 Render

128GB solo se recomienda para proyectos extremadamente complejos (edificios completos con paisajismo detallado)

## d) BenQ MA270U USB-C Power Delivery

VERIFICACIÓN PENDIENTE: No encontré especificaciones confirmadas del modelo MA270U

Los modelos BenQ profesionales típicamente incluyen USB-C con Power Delivery

Acción recomendada: Verificar ficha técnica específica del modelo en BenQ México

# 4. RECOMENDACIONES DE PSU Y CABLES

## PSUs Específicas Recomendadas para RTX 5090

## Cables y Conexiones

Cable requerido: 12V-2x6 (estándar ATX 3.1)

Compatibilidad: 12V-2x6 es retrocompatible con 12VHPWR pero más seguro

Adaptador incluido: RTX 5090 incluye adaptador 4x 8-pin a 16-pin, pero se recomienda cable nativo

ADVERTENCIA: NO usar adaptadores Molex-to-12VHPWR (riesgo de incendio)

## Marcas de PSU ATX 3.0/3.1 Disponibles en México

Tier 1: Seasonic, Corsair, ASUS ROG, EVGA (disponibles en Cyberpuerta, DDTech)

Tier 2: MSI, Thermaltake, Cooler Master (disponibles ampliamente)

Evitar: Marcas genéricas sin certificación 80+ o estándar ATX antiguo
