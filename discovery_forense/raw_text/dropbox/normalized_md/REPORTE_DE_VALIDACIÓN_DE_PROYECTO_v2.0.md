## 1. Idea Pura v1 (sin números) — Enfoque Híbrido C

U NAJIL JA es un proyecto de economía circular diseñado para la restauración ambiental de la Laguna de Bacalar. El concepto se basa en la extracción controlada de sedimentos acumulados que dañan el ecosistema, para luego darles un uso beneficioso y, en una segunda etapa, transformarlos en productos comerciales de valor agregado.

El proyecto se estructura bajo un Enfoque Híbrido C, que consiste en dos capas operativas:

Capa 1: Beneficial Use (Uso Beneficioso): Esta fase se enfoca en la ejecución de una obra de restauración ambiental, como la recuperación de playas o la estabilización de costas, utilizando los sedimentos dragados. El objetivo principal de esta capa es obtener los permisos regulatorios necesarios, demostrar la viabilidad operativa a gran escala, generar confianza con las autoridades y la comunidad, y manejar grandes volúmenes de material de manera eficiente.

Capa 2: Producto Comercial: Una vez que la Capa 1 está operativa, se desvía una fracción del flujo de sedimentos para ser procesada y convertida en productos comerciales. El enfoque inicial recomendado es la producción de materiales para uso civil, como sub-bases para carreteras o material de relleno estabilizado, debido a su menor fricción social y regulatoria en comparación con productos de consumo final como ladrillos o cerámicas.

Este enfoque de dos capas permite mitigar los riesgos regulatorios y de mercado, construyendo una base operativa sólida antes de escalar hacia productos de mayor margen pero mayor complejidad.

## 2. Metodología de Construcción Verificada

El proyecto se ejecutará en cuatro fases, diseñadas para validar progresivamente la viabilidad y escalar de manera controlada.

### Fase 0: Elegibilidad y Kill-Switch

Antes de cualquier inversión, se debe confirmar que no existen barreras regulatorias insuperables. Esto incluye una evaluación preliminar con SEMARNAT y autoridades locales para determinar si el concepto de "Beneficial Use" es viable en la Laguna de Bacalar. Si se detecta un "kill-switch" (una prohibición explícita o un requisito imposible de cumplir), el proyecto se detiene.

### Fase 1: Beneficial Use (Permisos + Volumen)

El objetivo de esta fase es ejecutar un proyecto de restauración a escala piloto o media. Esto permitirá:

Obtener los permisos de dragado y disposición de material.

Demostrar la capacidad de manejar volúmenes significativos de sedimento.

Generar datos operativos reales sobre costos y eficiencia.

Construir una relación de confianza con reguladores y la comunidad.

### Fase 2: Producto Comercial (Convertir Fracción del Flujo)

Con la operación de la Fase 1 estabilizada, se implementa una planta piloto para procesar una porción del sedimento extraído. El producto inicial será un material de construcción de bajo riesgo, como agregados para sub-base de carreteras, estabilizado con cal o cemento.

### Fase 3: Escalamiento

Si la Fase 2 demuestra ser técnicamente viable y comercialmente rentable, se procede a escalar la producción. Esto puede incluir la diversificación hacia otros productos (ej. ladrillos, cerámicas) o el aumento de la capacidad de la planta de procesamiento.
