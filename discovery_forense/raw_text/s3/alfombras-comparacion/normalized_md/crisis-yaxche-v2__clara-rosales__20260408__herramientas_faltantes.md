# Herramientas Faltantes Identificadas

## APIFY (No estaba en inventario)
- Plataforma de web scraping y extracción de datos a escala
- Actors disponibles para crisis política:
  - Facebook Pages Scraper: Scraping de páginas de FB (ej. Noticias Al Punto)
  - Instagram Scraper: Perfiles, posts, comentarios, hashtags, reels
  - All-in-One Social Media Scraper: Instagram, TikTok y más
  - Political News Event Monitoring & Clustering: Monitoreo de eventos políticos en tiempo real con clustering de noticias
  - Truth Social Scraper: Monitoreo político
- Capacidades clave: scraping masivo, monitoreo automatizado, APIs para integración
- Precio: Por uso (créditos)

## BRANDMENTIONS (No estaba en inventario)
- Plataforma de social listening y monitoreo de marca con IA
- 94% de precisión en análisis de sentimiento a nivel de entidad
- Monitoreo en tiempo real en 70+ fuentes (redes, blogs, noticias, foros)
- Tiene API propia para integración
- Notificaciones en tiempo real de menciones nuevas
- Análisis de sentimiento, tendencias, influencers
- Integrable con n8n y 61+ apps vía Integrately
- Planes desde básico hasta enterprise con API

## TAMBIÉN DESCUBIERTO: Manus Data APIs
- Twitter API (get_user_profile, get_user_tweets) - disponible en sandbox
- TikTok API (get_user_info, search_tiktok_video) - disponible en sandbox
- YouTube API (search, get_channel_videos, get_channel_details) - disponible en sandbox
- Reddit API (get_subreddit_hot_posts) - disponible en sandbox
Estas APIs están INTEGRADAS en el sandbox de Manus y no requieren credenciales externas.
