# Mapa Lógico de Formatos Legacy a SHELL

## Esencia a Transponer
Los formatos legacy (JSON, YAML, schemas) tienen una esencia que debe ser transpuesta al espacio SHELL, no como texto, sino como propiedades multidimensionales.

| Concepto Legacy | Propiedad en Espacio SHELL |
|---|---|
| Estructura | Espacio / Topología |
| Jerarquía | Profundidad (Eje Z) |
| Campos / Claves | Coordenadas / Nodos |
| Valores | Micropartículas |
| Relaciones | Relación Espacial / Vectores |
| Restricciones | Reglas de repulsión/atracción |
| Validación | Estabilidad de la configuración |
| Estados | Rotación / Color (frecuencia) |
| Tamaño de archivo | Densidad de la nube de polvo |
