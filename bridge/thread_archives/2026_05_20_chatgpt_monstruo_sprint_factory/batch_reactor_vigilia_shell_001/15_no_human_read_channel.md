# El Canal No Humano

## El Paradigma Anterior
Históricamente, obligamos a las IAs a pasarse el contexto usando el mismo formato que usamos los humanos para leer (lenguaje natural, JSON con claves semánticas en inglés/español). Esto es ineficiente.

## El Nuevo Paradigma (SHELL)
El canal IA↔IA se libera de la restricción de legibilidad humana. Si un humano intenta leer el Handoff Packet en formato SHELL, verá números, vectores o matrices aparentemente aleatorias. 

## Consecuencia
Solo la Unified Face tiene el trabajo de traducir el estado interno del Monstruo a lenguaje natural para Alfredo (T1). Todo el tráfico interno entre loops (el State Fabric y los Handoffs) eventualmente migrará a SHELL.
