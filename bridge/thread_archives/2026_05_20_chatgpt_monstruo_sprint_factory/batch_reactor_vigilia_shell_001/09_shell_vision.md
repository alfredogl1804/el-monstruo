# SHELL / Micropolvo Semántico

## Visión
El contexto IA↔IA no debe viajar como texto humano, JSON, YAML o schema final. Debe viajar como micropolvo semántico en un espacio matemático/multidimensional no humano-readable.

## Frases Doctrinales a Preservar
- "Sí viaja el contexto, pero es una nueva forma de contexto."
- "El contexto viaja como micropolvo semántico."
- "No se comprimen palabras. Se cambia la materia del contexto."
- "El significado vive en la configuración, no en la secuencia."

## Qué NO es SHELL
No reducir a TIC, PULSE, COSMOS, bytecode, puntero, diccionario, QR, Morse, JSON compacto, schema normal, prompt, resumen o RAG.
