# Selección de Input para Benchmark

## Criterios para el Corpus de Prueba (50k)
El texto de prueba debe ser altamente denso, contradictorio, lleno de reglas y dependiente del tiempo. No puede ser una novela.

## Corpus Propuesto
- Los 35 archivos del Thread Archive original.
- El log de decisiones T1.
- El modelo de Escalera de Autonomía.
- Los schemas del State Fabric.

## Preparación
1. Compilar el corpus en un solo string (IA A).
2. Transcribir (teóricamente) el corpus a espacio SHELL (IA B).
3. Evaluar los outputs.
