<td>Cambios Clave desde Marzo 2026</td>
<td>Desde marzo de 2026, Docker ha lanzado múltiples actualizaciones (v4.67.0 a v4.71.0). Los cambios clave incluyen: la introducción de Docker Model Runner (ahora deshabilitado por defecto pero con soporte para API compatible con Anthropic y vLLM en Windows con GPUs NVIDIA), mejoras en el MCP Toolkit (perfiles MCP, soporte para cliente Amazon Q y OAuth DCR), la adición de Gordon con memoria local persistente, y el fin del soporte para RHEL 8. También se han realizado múltiples correcciones de seguridad y mejoras en Enhanced Container Isolation (ECI).</td>
</tr>
