## L01 — Identidad Estratégica

Perplexity AI es un motor de respuesta conversacional impulsado por IA que prioriza la búsqueda web en tiempo real y la citación de fuentes. A diferencia de los LLMs tradicionales, Perplexity está diseñado para la recuperación de información precisa y actualizada, actuando como un "motor de respuestas" en lugar de un simple chatbot.

| Atributo | Valor |
|---------|-------|
| Empresa | Perplexity AI |
| Modelos soportados | Sonar Pro, Claude 3.5 Sonnet, GPT-4o, etc. |
| Tipo | Motor de respuestas / Agente de investigación |
| Diferenciador | Búsqueda web profunda en tiempo real con citas |
| Estado actual | Pro con Deep Research |

## L03 — Modelos y Contexto

Perplexity permite a los usuarios Pro elegir entre varios modelos de IA de vanguardia para procesar las búsquedas:

| Modelo | Proveedor | Uso Principal |
|--------|-----------|---------------|
| Sonar Pro | Perplexity | Búsqueda rápida y precisa (modelo propio) |
| Claude 3.5 Sonnet | Anthropic | Razonamiento complejo y escritura |
| GPT-4o | OpenAI | Tareas generales y multimodales |
| Otros | Varios | Opciones adicionales según disponibilidad |

El límite de contexto varía según el modelo seleccionado, pero Perplexity optimiza la recuperación de información para manejar consultas complejas.
