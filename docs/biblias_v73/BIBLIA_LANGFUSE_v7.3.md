<td>Cambios Clave desde Marzo 2026</td>
<td>Desde el 20 de marzo de 2026, Langfuse ha lanzado varias actualizaciones significativas. La versión v3.172.0 (29 de abril) introdujo un modo de filtro 'none' para etiquetas en la barra lateral y mejoras en la ingesta de metadatos de experimentos. La v3.171.0 (25 de abril) añadió la funcionalidad `source=ANNOTATION` para la API de scores y soporte para `gpt-5.5` en precios de modelos, además de mejoras de rendimiento en ClickHouse. La v3.170.0 (20 de abril) incluyó advertencias sobre caracteres especiales en `DATABASE_URL`, la emisión de métricas `.rate` y `.time` con etiquetas de shard, y la configuración de `ID Token signed response alg` para autenticación. También se mejoró la detección de la versión del SDK y se añadió soporte para activar evaluaciones en experimentos. Se implementaron nuevas opciones de frecuencia de exportación de almacenamiento de blobs (5 y 20 minutos) y una barra de acción flotante de selección múltiple en la UI. En cuanto a correcciones, se abordaron vulnerabilidades de seguridad (CSS injection, comparación de tiempo constante para API key), se mejoró la carga de esqueletos en tablas web y se corrigieron errores en el filtrado de scores y la visualización de JSONPath. Además, se añadió soporte para Langfuse Cloud Japan.</td>
</tr>
