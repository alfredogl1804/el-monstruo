## L01. Identidad y Propósito Core
**Nombre:** OpenAI o3
**Versión:** o3-pro (a partir de junio de 2025, la versión más reciente de o3)
**Propósito Principal:** Consultas complejas que requieren análisis multifacético; tareas visuales; programación; consultoría empresarial; ideación creativa; generación y evaluación de hipótesis (biología, matemáticas, ingeniería); tareas que requieren acceso a información actualizada, razonamiento extendido, síntesis y generación de resultados multimodales.

## L02. Capacidades Clave
Razonamiento avanzado en codificación, matemáticas, ciencia y percepción visual; análisis de imágenes, gráficos y diagramas; uso de herramientas (búsqueda web, análisis de archivos con Python, generación de imágenes); capacidad de agente para ejecutar tareas de forma independiente; manipulación de imágenes (rotación, zoom, transformación); alta precisión en tareas de percepción visual.

## L03. Limitaciones y Debilidades
Límites de uso: 50 mensajes por semana para usuarios de ChatGPT Plus (a partir de abril de 2025); límites de tokens (alrededor de 60-65K tokens por mensaje en la aplicación de chat); el artículo no menciona otras limitaciones específicas más allá de los límites de uso y tokens.
