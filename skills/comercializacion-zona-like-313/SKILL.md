# Zona Like 313 — Operational Memory & IA-First Blueprint

Este skill contiene la arquitectura comercial completa para vender las 313 butacas de Zona Like durante los 42 juegos de la temporada 2026 de los Leones de Yucatán.

## Premisa Fundamental (IA-First)

**"Los núcleos deben plantearse desde un momento de la IA en explosión. No pensar que se van a ejecutar esas estrategias de manera tradicional, con personas yendo a buscar a otras personas."**

El esfuerzo humano es mínimo. La IA genera el contenido, prospecta, segmenta, activa pautas y rescata inventario. Ticketlike (boletera propia) funciona como el CDP central.

## Cuándo usar este skill

1. **Diseño de Pautas:** Al crear campañas en Meta, Google o TikTok para Zona Like.
2. **Creación de Contenido:** Al redactar guiones, posts o anuncios sobre "Leoncio El Fugitivo".
3. **Automatización:** Al diseñar workflows (Make/n8n) para rescate de inventario o recompra.
4. **Análisis de Datos:** Al evaluar KPIs de ocupación y cruzar con el dashboard.
5. **Decisiones Estratégicas:** Al consultar el modelo financiero o el calendario de 42 juegos.

## Stack Tecnológico Autorizado

Cuando escribas código, prompts o configures herramientas para Zona Like, asume este stack:
- **Core:** Ticketlike (Motor de reservas y first-party data)
- **Pauta:** Meta CAPI + Advantage+, Google Performance Max, TikTok Smart Performance
- **Comunicaciones:** WhatsApp Business API
- **Prospección B2B:** B2B Rocket AI
- **Orquestación:** Make / Zapier / n8n
- **Generación:** GPT-5.4 / Claude 3.7 / Gemini 1.5 Pro / Midjourney / HeyGen

## Flujo de Trabajo Recomendado

1. Si te piden "hacer pauta para el martes": Lee `meta_estrategia_ia_first.md` (sección de Plantillas de Ocupación) para ver a quién le toca llenar ese día, y luego lee `leoncio_narrativa.md` para sacar el copy.
2. Si te piden "prospectar empresas": Asume que usarás `B2B Rocket AI` o similar, no un guion para call center.
3. Si te piden "rescatar un juego vacío": Revisa la sección "Pauta reactiva" y "Last-minute" en el blueprint principal.

**Recordatorio Anti-Autoboicot:** Nunca asumas que Zona Like es "vender boletos genéricos". Es vender **escasez premium (313 lugares)** usando una **narrativa serializada (Leoncio)** impulsada por **pauta algorítmica**.
