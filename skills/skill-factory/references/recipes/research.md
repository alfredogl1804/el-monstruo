# Recipe: Research & Analysis Skills

## Perfil del Dominio
- Complejidad típica: standard a advanced
- APIs frecuentes: Perplexity Sonar, Gemini (contexto largo), OpenAI
- Regulación: baja (excepto investigación médica/financiera)
- Investigación: siempre profunda (es el dominio mismo)

## Componentes Típicos

### Scripts Comunes
- Investigador web (usa Perplexity Sonar)
- Sintetizador de fuentes (usa Gemini para contexto largo)
- Generador de reportes (Markdown/PDF)
- Validador de fuentes (cross-reference)
- Extractor de datos estructurados (tablas, métricas)

### Referencias Comunes
- Fuentes confiables por subdominio
- Metodología de investigación
- Formato de citación
- Criterios de calidad de fuentes

### Templates Comunes
- Template de reporte de investigación
- Template de executive summary
- Template de comparativa/benchmark

## Modelo Recomendado por Tarea

| Tarea | Modelo |
|-------|--------|
| Búsqueda web en tiempo real | Perplexity Sonar |
| Síntesis de documentos largos | Gemini 3.1 Pro |
| Análisis crítico | Claude Sonnet 4.6 |
| Generación de reporte final | GPT-5.4 |
| Verificación de datos | Perplexity Sonar (segunda pasada) |

## Quality Gate Específico
- Todas las afirmaciones clave deben tener fuente
- Cross-validation con al menos 2 fuentes independientes
- Fecha de los datos verificada (no obsoletos)
- Sesgo detectado y documentado
- Executive summary presente

## Anti-patrones
- Citar una sola fuente como verdad absoluta
- No verificar fechas de publicación
- Mezclar opinión con hechos sin distinción
- Reportes sin estructura (wall of text)
- Ignorar fuentes en idiomas distintos al inglés
